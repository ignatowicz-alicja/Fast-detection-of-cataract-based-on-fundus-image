# MATLAB toolboxes

The pipeline may require the following MATLAB products/toolboxes, depending on enabled stages:

- MATLAB
- Image Processing Toolbox
- Statistics and Machine Learning Toolbox
- Deep Learning Toolbox
- Deep Learning Toolbox Model for ResNet-50 Network, if `RESNET50_TL` is enabled
- Deep Learning Toolbox Model for MobileNet-v2 Network, if `MOBILENETV2_TL` is enabled
- Parallel Computing Toolbox, optional, for GPU execution

The classical LBP/HIST + SVM/RF stages can be run without enabling the deep learning stage.
